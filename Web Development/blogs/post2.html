Python Physics Engine (Work in Progress)

Project Status: In Development • Last Updated: April 2025

This project aims to build a simple 2D physics engine in Python, focusing on collision detection, response, and basic simulation of physical systems. The goal is to create an educational tool for understanding the fundamentals of physics engines.

Note: This project is currently a work in progress. Features and documentation will be updated regularly as development continues.

Planned Features

  • 2D rigid body simulation
  • Basic collision detection (AABB, circles)
  • Collision response (elastic/inelastic)
  • Gravity and simple forces
  • Interactive visualization (coming soon)

Current Status

The core engine is under development. At this stage, the focus is on building a robust foundation for object management and collision detection. Visualization and user interaction will be added in future updates.

Stay tuned for updates and demos!
